git使用方法

发表于2023-02-08|更新于2023-02-16|教程git
|字数总计:380|阅读时长:1分钟|阅读量:

之前使用git都是用的GUI，用鼠标点就行了，操作简单，但是并不是很好用，很多功能都没有。想用Bash，但是又不知道命令，所以想着学习一下git的基本命令。水平有限，请见谅。

Git使用

创建仓库

  • git init:初始化一个仓库
  • git init {fileName}:新建名文件并初始化仓库
  • git clone {url}:通过url克隆仓库到本地
  • git clone {url} {fileName}:新建文件并从url克隆仓库到本地

查看信息

  • git config:配置邮箱和
  • git config --list:查看所有配置信息
  • git add:添加文件到缓存
  • git status:查看文件状态
  • git commit -m {message}:提交命令
  • git commit --amend:替代上一次提交
  • git log:显示当前分支的版本历史
  • git reset --hard{标识号}：撤销标识号之后的更改
  • git reflog:查看所有操作记录
  • git diff --shortstat “@{0 day ago}”:显示今天写了多少行代码

git版本管理

本地分支

  • git branch:列出所有本地分支
  • git branch -r:列出所有远程分支
  • git branch -a:列出所有分支
  • git branch {branchName}:新建分支
  • git checkout {branchName}:切换分支
  • git checkout -b {branchName}:创建并切换分支
  • git merge {branchName}:合并分支到当前分支
  • git branch -d {branchName}:删除分支

远程分支

  • git clone {url}:克隆远程仓库
  • git pull:拉取远程仓库
  • git push:推送远程仓库
  • git push --force:强行推送到远程仓库
  • git fetch {url}:下载远程仓库的所有变动
